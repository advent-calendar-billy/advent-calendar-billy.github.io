# Hand-Drawn Aesthetic Guide

## Color Palette

### Base Colors
- **Paper Background**: `#f5f0e1` (aged cream)
- **Grid Lines**: `rgba(0,0,0,0.03)` (subtle)
- **Ink Black**: `#333` (not pure black, softer)
- **Pencil Gray**: `#666`

### Accent Colors (colored pencil style)
- **Red Accent**: `#e74c3c` (slightly muted)
- **Blue Accent**: `#3498db`
- **Yellow Accent**: `#f1c40f`
- **Green Accent**: `#27ae60`

## Typography

### Primary Font
**Finger Paint** - Used for:
- Character names
- VS text
- Menu headers
- Power names

### Secondary Font
**Architects Daughter** or **Patrick Hand** - Used for:
- Body text
- Stats
- Descriptions

### Accent Font
**Permanent Marker** - Used for:
- Impact moments
- Critical hits
- Special effects text

## Visual Effects

### Line Style
- Lines should appear hand-drawn, slightly wobbly
- Variable stroke width (thicker at pressure points)
- Occasional "overshoot" at corners
- Cross-hatching for shadows

### Character Rendering
```
┌─────────────────────────────────┐
│   SKETCH STYLE ELEMENTS:        │
│                                 │
│   ○ Visible construction lines  │
│   ○ Slightly asymmetric shapes  │
│   ○ Rough, organic edges        │
│   ○ Doodle-style details        │
│   ○ ? marks for unknown chars   │
└─────────────────────────────────┘
```

### Power Effects
- Comic book style: "POW!", "WHAM!", "ZAP!"
- Hand-drawn explosion shapes
- Squiggly motion lines
- Stars and spirals for impact

### UI Elements
- Buttons look like drawn rectangles with rounded corners
- Health bars with sketchy edges
- Timer in a hand-drawn circle
- Score displayed on "torn paper" elements

## Animation Principles
- Slightly jittery/boiling effect (frames vary slightly)
- Squash and stretch exaggerated
- Motion blur represented by multiple sketch lines
- Impact frames with heavy ink splatter

## Background Elements
- Graph paper grid (25px squares)
- Coffee stain rings (optional)
- Margin lines on sides
- Hole punch marks at top
- Spiral binding suggestion on left edge
